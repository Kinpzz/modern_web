#####mole
mole游戏使用的是input radio完成游戏，与作业要求一致。<br />
当游戏结束的时候通过点击Start Game可以开始游戏，游戏开始之后点击Pause Game可以暂停游戏，暂停游戏之后点击Continue Playing可以继续开始游戏，在下边的状态栏显示游戏状态。
